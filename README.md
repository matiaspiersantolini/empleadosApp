# empleadosApp

Servicio Rest en el cual se pueden ingresar datos de los empleados de una empresa, consultarlos, actualizarlos y/o borrarlos.

-Tecnologia Java 8

-Framework Spring Boot

-Base de datos: Postgre SQL

-Documentada con Swagger.
